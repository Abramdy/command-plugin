Spring Boot Command Plugin
--------------------------

Add this dependency in your ```pom.xml```

```xml
<dependencies>
  ...
  <dependency>
   <groupId>com.blibli.oss</groupId>
   <artifactId>command-plugin</artifactId>
   <version>...</version>
 </dependency>
 ...
</dependencies>
```
